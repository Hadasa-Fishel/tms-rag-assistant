## Quick Reference

### Initialize Workflow
```python
from rag_workflow import RAGWorkflow, RAGConfig

config = RAGConfig()  # Loads structured_db.json automatically
workflow = RAGWorkflow(config)
```

### Run a Query
```python
result = await workflow.run(input="List all technical decisions")
```

### Check Results
```python
print(f"Response: {result['response']}")
print(f"Method used: {result['metadata']['retrieval_method']}")
print(f"Confidence: {result['confidence_score']:.2f}")
print(f"Sources: {result['metadata']['sources']}")
```

### Batch Processing
```python
import asyncio

queries = ["Query 1", "Query 2", "Query 3"]
tasks = [workflow.run(input=q) for q in queries]
results = await asyncio.gather(*tasks)
```

## Troubleshooting

### "structured_db.json not found"
```bash
python data_extractor.py
# Run this first to generate the file
```

### "No results found"
- Try rephrasing your question
- Use explicit keywords: "list", "all", "latest"
- Check that structured_db.json has 160 entities

### "I want to batch process 100 queries"
```python
# Use asyncio.gather() for parallel processing
import asyncio
tasks = [workflow.run(input=q) for q in queries]
results = await asyncio.gather(*tasks)
```
